Question-1: What are the different ways to select element in the DOM?


Answer: We can select elements in the DOM using getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll.


Question-2: What is the difference between innerHTML, innerText, and textContent ?


Answer: innerHTML returns or sets the HTML content inside an element, allowing HTML parsing. innerText returns or sets only the visible text, ignoring hidden elements. textContent returns or sets all text, including hidden elements, without parsing HTML.


Question-3: What is event delegation in the DOM?


Answer: Event delegation means letting the big parent box listen for clicks instead of each small box inside it. This way, even new small boxes added later still work


Question-4: What is event bubbling in the DOM?


Answer: Event bubbling means when we click a small box, the message goes up to the bigger boxes around it, one by one, like bubbles floating up in water


Question-5: How do you create, add, and remove elements using JavaScript?


Answer: we can create an element using document.createElement(), add it with appendChild() or append(), and remove it using removeChild() or remove().
